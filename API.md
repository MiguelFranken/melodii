# API

This file lists all possible osc-messages which the server can handle.

## Drums

### Kick
<details><summary>Plays a drum kick note</summary>
<p>

```
/drums/kick 
```
Arguments: 
```
[
    { s,note },  // Expects a note as string 
    { i,velocity },  // Expects the velocity of the note as float
]
```

</p>
</details>
